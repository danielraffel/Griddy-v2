Griddy Third-Party Credits and Licensing

This page documents the direct third-party components observed in the local Griddy macOS Release build as of March 10, 2026. It is intended to ship with the product and to refresh ~/Library/Caches/Griddy/Griddy_Licenses.html whenever the acknowledgements button is opened from the plugin.

Last updated: March 10, 2026

JUCE Framework

Source: juce-framework/JUCE
License: AGPL v3.0 and commercial licensing
Used for: plugin framework, audio and MIDI I/O, host integration, windowing

Griddy uses JUCE as its primary application and plugin framework. The local project is configured against JUCE 8.0.12 in the build configuration.

Mutable Instruments Grids

Original project: Mutable Instruments Grids
License: GNU General Public License v3.0
Copyright: Emilie Gillet

Griddy incorporates Grids-derived rhythm generation code and pattern data as the musical core of the plugin.

Additional Bundled Notices

Confirmed from: the local JUCE LICENSE.md dependency manifest and the local Visage source tree

The current Griddy macOS source tree also carries bundled upstream components through JUCE and Visage beyond the top-level items listed above.

  • stb_image_resize2 is included through Visage image handling and is distributed as public domain or MIT-licensed code.
  • JUCE's local dependency manifest lists AudioUnitSDK (Apache 2.0), FLAC (BSD), Ogg Vorbis (BSD), jpeglib (Independent JPEG Group), pnglib and zlib (zlib), HarfBuzz (Old MIT), and SheenBidi (Apache 2.0).
  • The same JUCE tree also carries format SDK notices for VST3 (MIT), LV2 (ISC), and AAX (Avid AAX license or GPLv3) for formats built from this source tree.

Those entries come from upstream frameworks used by Griddy rather than separate top-level libraries unique to this project, but they are worth disclosing because the older cached page did not mention them.

The previous cached acknowledgements page was missing at least the direct rendering-stack dependencies bgfx, bimg, bx, freetype, and the bundled Visage fonts. This refreshed page now also calls out the additional bundled notices that were visible in the local JUCE and Visage trees as of March 10, 2026.
